# Site Officiel d'Orphée NYNY

Site web professionnel pour Orphée NYNY, artiste, chanteuse, compositrice et interprète gabonaise.

## 🎵 À propos

Ce site présente l'univers artistique d'Orphée NYNY, une artiste talentueuse originaire de Bitam, au Gabon. Le site offre une expérience immersive permettant de découvrir sa musique, son parcours, et de rester informé de ses actualités et événements.

## 🌟 Fonctionnalités

### Pages principales
- **Accueil** : Page d'accueil avec hero section, biographie courte, lecteur vidéo, dernières créations et événements à venir
- **Biographie** : Parcours détaillé de l'artiste avec photos et texte enrichi
- **Discographie** : Catalogue complet des singles et albums avec liens vers les plateformes de streaming
- **Clips & Médias** : Galerie de vidéos YouTube avec lecteur intégré
- **Événements** : Liste des concerts et événements à venir et passés
- **Galerie** : Collection de photos avec système de filtrage par catégorie et lightbox
- **Services** : Présentation des prestations professionnelles (concerts, compositions, collaborations, etc.)
- **Contact** : Formulaire de contact avec informations de contact et liens réseaux sociaux

### Pages légales
- **Mentions légales** : Informations légales obligatoires
- **Politique de confidentialité** : Gestion des données personnelles et RGPD

## 🛠️ Technologies utilisées

- **Framework** : Next.js 16.1.3
- **React** : 19.2.3
- **Styling** : Tailwind CSS
- **Icons** : React Icons
- **Fonts** : Google Fonts (Baskervville, Poppins)
- **Animations** : CSS transitions et Tailwind animations

## 🎨 Design

Le site utilise une palette de couleurs élégante :
- **Primaire** : `#2f0f09` (brun foncé)
- **Secondaire** : `#F2E1CA` (beige clair)
- **Accent** : `#4a1810` (brun moyen)

### Typographie
- **Titres** : Baskervville (serif élégante)
- **Corps de texte** : Poppins (sans-serif moderne)

### Caractéristiques visuelles
- Design responsive (mobile-first)
- Animations fluides et micro-interactions
- Images en parallaxe (effet fixed background)
- Navigation sticky avec effet de masquage au scroll
- Cartes avec effets hover 3D
- Lightbox pour la galerie photos
- Lecteur vidéo YouTube intégré

## 🚀 Installation et développement

### Prérequis
- Node.js 18+ 
- npm ou yarn

### Installation

```bash
# Cloner le repository
git clone [url-du-repo]

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

Le site sera accessible sur `http://localhost:3000`

### Build de production

```bash
# Créer le build de production
npm run build

# Lancer le serveur de production
npm start
```
